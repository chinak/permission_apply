- 使用 React Context API 进行跨组件状态共享，并通过自定义 Hook（如 `useAppContext`）封装访问逻辑，在 Provider 未注入时抛出错误以保证类型安全。
- 页面间临时数据传递依赖 `react-router` 的 `location.state`，用于在跳转时携带审批状态、拒绝原因、mockExchangeIds 等一次性参数。
- UI 样式遵循 Tailwind CSS 约定，广泛使用 `cn` 工具函数处理条件类名拼接，保持组件样式的灵活性与一致性。
- 导航与面包屑采用数据驱动模式：通过 `NAV_GROUPS`、`GROUP_BREADCRUMB_MAP` 等常量集中声明菜单结构与路径标签，避免在组件内硬编码分支判断。
