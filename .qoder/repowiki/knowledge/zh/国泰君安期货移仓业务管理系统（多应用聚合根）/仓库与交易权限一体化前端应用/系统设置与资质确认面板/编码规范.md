- 所有表单输入（Select、Textarea、input）均采用受控模式，通过 state + onChange 驱动视图更新。
- 弹窗类组件统一采用组合式 API 设计，通过 open/onOpenChange 属性控制显隐，避免在组件内部持有打开状态。
- 样式统一使用 Tailwind CSS，遵循 slate/blue 配色体系及 rounded-sm 圆角规范，按钮 hover 态通过 bg-blue-50/text-blue-700 表达。
- 图标统一从 lucide-react 导入并使用 className 指定尺寸（w-4 h-4 / w-6 h-6），不直接使用 SVG 字符串。
