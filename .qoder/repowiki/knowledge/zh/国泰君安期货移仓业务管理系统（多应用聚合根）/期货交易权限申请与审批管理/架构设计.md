- **技术栈**: React 18 + TypeScript, Vite 6, Tailwind CSS 4, React Router 7。
- **UI 架构**: 采用 shadcn/ui (Radix UI) 组件库构建原子化 UI，结合 MUI Icons 和 Lucide React 图标。
- **状态管理**: 使用 React Context (`AppContext.tsx`) 集中管理全局业务状态（如风险等级 C3-C5、资金档位、已选交易所），实现跨页面数据共享。
- **路由设计**: 基于 `react-router` 的嵌套路由结构，`Layout` 组件作为根布局包含侧边栏导航，子路由涵盖首页、申请表单、详情查看及审批列表。
- **业务逻辑分层**: 核心逻辑集中在 `Home.tsx`（权限选择与初步校验）和 `SubmitForm.tsx`（多场景申请材料提交），通过 `ConfigPanel` 提供开发态的状态模拟能力。
