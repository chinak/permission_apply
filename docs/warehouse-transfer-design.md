### 移仓信息区
| 字段 | 输入方式 | 业务规则 |
|------|----------|----------|
| 移出/移入方向 | 单选 | 移出我司 / 移入我司 |
| 目标期货公司 | 输入框 | 对方期货公司全称 + 会员号 |

| 移仓手数 | 数字输入 | 上期所需为交割单位整数倍 |

| 移仓类型 | 单选 | 跨期货公司转移 / 实控组账户内转移 |
| 实控关系组号 | 输入框 | 选择实控组内转移时必填 |
| 是否套保合约 | 单选 | 是 / 否，大商所/上期所套保移入需已有套保额度 |

### 风险确认区
- 移仓前一日结算后可用资金不为负
- 移仓账户交易编码状态正常
- 不存在被监管限制或异常情况
- 已阅读并同意《交易所移仓业务风险揭示书》

### 附件区
- 上传已盖章的《移仓业务申请表》扫描件
- 实控关系证明文件（如适用）
- 套保资格/额度证明（如适用）

### 操作按钮
- 保存草稿
- 提交申请

## 二、总部审核界面设计

### 顶部摘要卡
| 字段 |
|------|
| 流水号 |
| 客户名称 / 资产账号 |
| 交易所 / 合约 / 手数 |
| 移仓方向 |
| 目标期货公司 |
| 申请状态 |

### 审核工作区（Tab 分块）
1. **客户申请信息** — 表单只读展示
2. **账户风控校验**
   - 交易编码状态
   - 可用资金（移仓前一日结算后）
   - 持仓数量
   - 风险度
   - 监管限制 / 异常状态
3. **交易所规则校验**
   - 合约是否在允许移仓时间范围内
   - 手数是否满足交割单位整数倍（上期所）
   - 实控关系是否生效（上期所需提前 10 个交易日）
   - 套保资格 / 额度校验
4. **附件资料**
   - 申请表扫描件预览
   - 其他证明文件
5. **操作日志 / 流程记录**
   - 客户提交
   - 业务单位初审
   - 运营中心复核
   - 交易所审批结果

### 审核操作
- 通过 → 提交交易所
- 驳回 → 选择/填写原因，退回业务单位/客户
- 暂缓受理 → 因风险度 > 100% 或极端行情
