# 国泰君安期货交易所移仓业务操作指引

## 第一章 业务概述

**第一条** 为规范公司交易所移仓业务操作流程，防范业务风险，根据《郑州商品交易所期货结算管理办法》、《大连商品交易所结算管理办法》、《上海期货交易所交割月移仓业务指引》相关要求，结合我司具体情况，特制定本指引。

**第二条** 交易所移仓业务是指：客户满足相应条件的持仓，经客户申请、期货公司和交易所审批同意，可以从一家期货公司（移出方）转移至另一家期货公司（移入方），或在同一实控组内不同账户间转移。已开展移仓业务的交易所可移仓类型如下：

1. **郑州商品交易所**：同一客户在不同期货公司会员之间进行合约持仓及持仓保证金的转移；
2. **大连商品交易所**：同一客户在不同期货公司会员之间进行合约持仓及持仓保证金的转移；同一组实际控制关系账户内，单位客户、非期货公司会员或境外特殊非经纪参与者的期货及期权持仓在关系组账户之间进行合约持仓的转移（非期货公司会员或境外特殊非经纪参与者仅能移出）；
3. **上海期货交易所**：同一组实际控制关系账户内，境外单位客户或境外特殊非经纪参与者向境内一般单位客户或非期货公司会员进行特定品种的持仓转移（品种范围以交易所发布为准）。
4. **其他交易所**：暂无该业务

**第三条** 我司客户申请办理交易所移仓业务须满足的条件：

1. 客户账户应当活跃，且移仓交易所交易编码处于正常状态；
2. 客户账户应满足移仓合约的适当性限制要求；
3. 客户账户保证金满足移仓业务客户风险控制要求，且移仓前一日结算后可用资金不为负；
4. 若客户存在质押业务，客户账户在移仓后需满足质押配比的资金要求；
5. 对于大连商品交易所、上海期货交易所实际控制关系组内的套保合约移入申请，客户需已获得相应套保资格或额度；
6. 对于大连商品交易所移仓，允许办理时间为合约上市日起至合约最后交易日前一日；
7. 对于上海期货交易所移仓，允许办理时间为移仓合约的交割月第一个交易日起至最后交易日前第四个交易日（含当日）止；
8. 对于上海期货交易所移仓，移出方和移入方应在移仓前至少提前10个交易日确认实控关系已生效；
9. 对于上海期货交易所移仓，申请移仓手数应当为交割单位对应手数的整数倍；
10. 客户不存在被监管限制或其他异常情况。

## 第二章 职责分工

**第四条** 客户所在业务单位负责协助运营中心办理移仓业务，IB客户由IB业务服务部负责协助，具体职责如下：

1. 审核客户是否满足交易所移仓业务办理要求、协助客户填写《国泰君安期货有限公司移仓业务申请表》（附件一），并确保文件内容的清晰、完整及可追溯性；
2. 对客户提交的移仓业务申请表进行初审，并做好风险揭示工作；
3. 负责业务办理过程中客户对应持仓等关键信息变化的信息同步工作；
4. 如因当日办理移仓业务导致客户出现结算后可用资金为负的情况，配合做好公司下一交易日采取提高保证金、限制出金、冻结资金、强行平仓等一种或多种措施的客户解释工作。

**第五条** 运营中心负责交易所移仓日常业务管理与办理，具体职责包括：

1. 负责审核业务单位或IB业务服务部提交的业务申请表；
2. 负责对申请移仓客户账户的资金和持仓相关风险评估和管理；
3. 负责向交易所提交移仓业务申请，跟踪业务办理进度，向业务单位或IB业务服务部反馈业务办理结果；
4. 日终结算时根据交易所下发的结算文件，完成客户在我司移仓结算工作。

## 第三章 交易所移仓业务的办理

**第六条** 客户申请办理交易所移仓业务的，按如下流程办理：

1. 业务单位及IB业务服务部核查客户移仓资质，对于符合移仓业务要求的，方可提交移仓业务申请；
2. 客户需填写《国泰君安期货有限公司移仓业务申请表》（附件一），由客户所属业务单位或IB业务服务部向运营中心提交已盖公章确认的电子版申请表，经由负责人审批同意后，明确具体的业务办理事项，与运营中心协商确定移仓日期，准确提供客户在另一家会员处的持仓、资金等信息。原则上移仓业务的申请截止时间为申请办理当日的14:00之前；
3. 运营中心接到移仓业务申请后，确定具体业务对接人，进行移仓日的预结算等压力测试，核查移仓客户的资质情况，确定客户实际控制关系及套保资格等信息，盘中提交交易所办理。对于上一交易日结算后客户出现交易所透支风险即交易所风险度超过100%的，或市场发生极端行情等特殊情况，运营中心有权延缓受理移仓业务申请；
4. 对于因发生交割、行权履约、期转现等业务，导致持仓发生变化的情况，即便经期货公司会员提交了移仓业务申请并经过交易所审批，也可能导致客户移仓业务失败或移仓结果和实际申请内容有出入；
5. 运营中心完成客户移仓业务办理后，反馈申请单位，申请单位反馈客户。
6. 各业务单位在办理移仓业务过程中，可就移仓业务申请表电子文件形式进行审核、存档及业务处理。

**第七条** 对于交易所收取的移仓手续费，运营中心从客户账户中代扣代缴。如需另行加收，业务单位或IB业务服务部须与客户协商一致并签订收费协议，在收取日结算前通知运营中心，运营中心审核通过后，从客户期货保证金账户内收取该笔费用。
